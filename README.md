# 🛒 Zepto SQL Analysis

## 📌 Overview

This project analyzes Zepto product data using PostgreSQL. The analysis includes data cleaning, exploratory data analysis, and business-focused SQL queries to generate meaningful insights from product, pricing, and inventory data.

## 🛠 Tools Used

- PostgreSQL
- SQL

## 📚 SQL Concepts Used

- SELECT
- WHERE
- GROUP BY
- HAVING
- ORDER BY
- Aggregate Functions
- CASE Statements
- DISTINCT
- LIMIT

## 📈 Business Questions Solved

- Top 10 products with the highest discounts
- High MRP products that are out of stock
- Estimated revenue by category
- Products with high MRP but low discounts
- Categories with the highest average discount
- Price per gram analysis
- Product weight categorization
- Total inventory weight by category
